# 🤖 AI-Powered API Documentation Generator

An AI-powered tool that automatically generates API documentation for Express.js applications. It scans Express route files, uses Google Gemini AI to understand each endpoint, and generates interactive Swagger/OpenAPI documentation.

## ✨ Features

- 📂 Automatically scans Express.js route files
- 🤖 Uses Gemini AI to analyze API endpoints
- 📝 Generates API descriptions automatically
- 🌐 Creates interactive Swagger/OpenAPI documentation
- ⚡ Eliminates the need for manual API documentation

## 🛠️ Tech Stack

- Node.js
- Express.js
- Google Gemini AI
- Swagger UI
- OpenAPI

## ⚙️ How It Works

1. Scan Express route files.
2. Send each route to Gemini AI.
3. AI analyzes the endpoint.
4. Generate an OpenAPI specification.
5. Display the documentation in Swagger UI.

## 💡 Why It's Helpful

- Saves development time by automating API documentation.
- Keeps documentation aligned with the actual route code.
- Reusable across multiple Express.js projects.
- Demonstrates practical use of AI in backend development.
